In order to run the CYTEZEN Web App you must do the following:
- Set up your Eclipse development environment as outlined here: http://code.google.com/appengine/docs/java/gettingstarted/installing.html
- Create a sample developer app on Facebook using your own credentials.
- Copy the App ID, Key, and Secret into the FacebookUtil class.
- Build and run the CYTEZEN Web App from Eclipse.
- Open up a web browser and navigate to: http://localhost:8888/
